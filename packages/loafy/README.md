## Quick Start

### Using npx (Recommended)

```bash
npx create-loafy@latest my-app

# or...

bun create loafy@latest my-app
```

### Using global installation

1. **Install Loafy globally**:
   ```bash
   npm install -g loafy
   # or
   pnpm add -g loafy
   # or  
   yarn global add loafy
   # or
   bun add -g loafy
   ```

2. **Create a new project**:
   ```bash
   loafy init my-app
   ```

3. **Navigate to your project**:
   ```bash
   cd my-app
   ```

4. **Start developing**:
   ```bash
   npm run dev
   # or
   pnpm dev
   # or
   yarn dev
   # or
   bun dev
   ```
